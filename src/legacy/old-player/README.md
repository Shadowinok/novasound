# Old Player Snapshot

This folder stores a rollback snapshot of the working player version.

## Files

- `PlayerContext.old-player.jsx`
- `PlayerBar.old-player.jsx`

## Fast rollback

Replace:

- `src/context/PlayerContext.jsx` with `PlayerContext.old-player.jsx`
- `src/components/PlayerBar.jsx` with `PlayerBar.old-player.jsx`

Then rebuild frontend.
